赤寒莊特攻介紹

名稱

指令

 條件

備註
採藥 cmd search 　

1. 只有在赤寒莊藥園或是幫派藥園才能採藥

2. 不管成功或失敗都會增加藥理值

3. 藥材是煉丹材料，煉丹會消耗藥理值

4. 藥理上限 Exp/100+1000

5. cure等級影響採藥時間長短

6. cold-cure等級影響採藥成功率
藥材箱 cmd collect 解謎

1. 一格最初儲存數量為一百，可擴充至一千

2. 最初儲存格數為六格，可擴充至十六格
主動解定 cmd delbusy

coldforce Lv> 330

cure Lv >= 150

force Lv > 300

exp > 2000萬

coldforce越高，使用時扣的內力越少
