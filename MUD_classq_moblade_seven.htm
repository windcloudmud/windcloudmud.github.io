【ask yo about 七大限】

你恭敬地向蚩尤請教關於「七大限」的問題。
蚩尤說道：七大限乃我手執「虎魄」所創，其中蘊含無限悲愴的刀意，所以破壞極強，常人難以駕馭。

　

【ask yo about 虎魄】

你恭敬地向蚩尤請教關於「虎魄」的問題。
蚩尤說道：虎魄乃「細虎」之脊骨，混合天外異妖之血所煉，江湖因此傳言我「屠虎煉刀」。

　

【ask yo about 細虎】

你恭敬地向蚩尤請教關於「細虎」的問題。
蚩尤說道：細虎乃我自幼所依的虎娘「大媽」之子，與我情同手足，忠心不二。

　

【ask yo about 屠虎煉刀】

你恭敬地向蚩尤請教關於「屠虎煉刀」的問題。
蚩尤說道：江湖傳言我殺子餵妖，屠虎煉刀，是因為我曾放逐自我，跟隨異妖，為其謀食，苟且偷生...
蚩尤說道：一日我子前來尋我，不料我無意中竟將其獻給異妖所食，細虎通靈遂與異妖纏鬥...
蚩尤說道：等我發現時，我子已被異妖吸成人乾，一時悲憤攻心，將異妖開腸剖肚，細虎亦噬咬其頭...
蚩尤說道：而正當我因無心弒子而感到痛心疾首之時，細虎忽然背部隆起，痛苦非常...
蚩尤說道：我為減其痛楚，遂忍痛抽其脊骨，斃其命...
蚩尤說道：豈料抽出來的竟已成為一柄鋒芒凜冽，刀身剔透的神兵，正是虎魄。

　

【ask yo about 大媽】

你恭敬地向蚩尤請教關於「大媽」的問題。
蚩尤說道：我自幼喪親，體內又留有虎獸血脈，與生俱來就擁有和老虎溝通的異能...
蚩尤說道：加上久別人世，促成我深交的，便是一對老虎母子，即大媽和細虎...
說到此處...蚩尤忽然臉泛青光，怒意大現。
蚩尤說道：口胡～此「奪母之仇」我必報不可啊！！！

　

【ask yo about 奪母之仇】

你恭敬地向蚩尤請教關於「奪母之仇」的問題。
蚩尤說道：我自幼便將大媽視為母親，但大媽卻被巨獸所殺，化為紫晶虎骨...
蚩尤說道：我為取得大媽虎骨，放棄誅殺巨獸的機會，豈料...
蚩尤說道：豈料...「雷武」，沒錯...就是雷武這天殺的賊畜生奪走我的一切！！！

　

【ask yo about 雷武】

你恭敬地向蚩尤請教關於「雷武」的問題。
蚩尤說道：雷武...這個賊雷武...是個表裡不一的衣冠禽獸...
蚩尤說道：他假意與我結拜，卻伺機奪走我的財富權力，崇高地位以及幸福的家庭溫暖...
蚩尤說道：一夜之間，我失去一生所有，更背信任的兄弟出賣，這種傷痛，非筆墨所能形容...
蚩尤說道：他甚至奪走大媽的紫晶虎骨，還將其煉成「無上虎魄」，助其大業。

　

【ask yo about 無上虎魄】

你恭敬地向蚩尤請教關於「無上虎魄」的問題。
蚩尤說道：無上虎魄是大媽的遺骸，我定要將其收回，並好好安存...
蚩尤說道：無奈現在我已身為邪龍塔內三樓之主，實在無法抽身親自取回...
蚩尤說道：今日見你一身魔性與雷勁，看來是有能耐繼承七大限的破壞之能...
蚩尤說道：這樣吧...你替我奪回無上虎魄，我便將七大限的奧義傳授於你...
蚩尤說道：你若是答應這個交換條件，便與我執手為盟(hold hand)吧！

　

【hold hand】
你決定與蚩尤執手為盟。
蚩尤說道：既然你已答應了我，我便告訴你雷武的所在之處...
蚩尤說道：由於他身為英雄盟主，正與武林正派聯繫要攻上赤煉邪龍塔之事宜...

蚩尤說道：我的眼線告知我，雷武現在正於北冥世家內作客。
蚩尤話剛說完，旋即雙手畫圓，創造了一個時空裂縫。
蚩尤說道：在此先祝你馬到功成，一切拜託你了！！！
