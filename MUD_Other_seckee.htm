＊本篇說明

　　 靈氣可透過解謎獲得(每種靈氣只能解一次)，新獲得的靈氣會覆蓋掉身上原本的靈氣，因此必須注意靈氣的解謎順序。

　　每種靈氣都有兩種用途，一種是用來殺敵，另一種則是用來提升自己的能力，而不同的靈氣所發出來的作用皆不相同，且靈氣強弱不同，威力也不同。

　

　

＊關於靈氣

【關於靈氣的功能可以參考下表】

種類 功能 解謎連結

【關於靈氣的相關指令可以參考下表】

【注意！靈力≠靈氣】

指令 功能
cmd usekee 戰鬥中使用靈氣(會使靈氣值歸零)
cmd special 非戰鬥中使用靈氣(會使靈氣值歸零)
respirate <數字>

精力轉靈力，靈力破表時可增加靈力上限，

靈力最大值由基本法術等級決定
cmd gather

靈力轉靈氣，靈氣破表時可增加靈氣上限，

最大值由靈力的多寡決定
exert <數字> reseckee 內力轉靈氣，只能補滿靈氣，無法突破上限

　

【注意！新獲得的靈氣會覆蓋掉身上原本的靈氣】

【一般我們會建議先解青龍，因為青龍最沒有用】

【雖然可以增加威力，但是根本感覺不出來，所以先解青龍】

　

【第二個一般會解玄天，因為玄天可以增加內功有效等級】

【配合穿滿內功裝備可以衝高內力上限，對初期要衝高內力相當好用】

　

【最後通常我們會換成雷虎，增加氣血上限可以讓戰鬥有更高的勝算】

【但是血太多也不是沒有壞處，如果你的門派有加血招還能騎馬】
【再加上雷虎可能會讓氣血跟內力的比例失衡，導致吸氣吸不回來】

　

【所以一般不會有吸氣問題的狀況順序是：青龍－＞玄天－＞雷虎】

【有吸氣問題或是非常重內力的狀況順序是：青龍－＞雷虎－＞玄天】

　

【關於如何衝高靈氣值】

１. 先把基本法術(magic)學滿

２. 使用 respirate 將靈力練到最大值

３. 使用 cmd gather 將靈氣練到最大值

４. 靈氣練滿後，平常戰鬥使用 exert reseckee 補充靈氣值(補比較快)

５. 確定你的靈氣種類不會再改變時，可以解謎增加靈氣最大值
