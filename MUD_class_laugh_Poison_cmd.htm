冥蠱魔教公會指令介紹
名稱 指令 條件 備註
吸毒納蠱 cmd suck <毒的名字> poison 無

１. 增加體內毒素

２. 許多招式與其施放有關

丟毒

 cmd throw <毒的名字> to <mob> 無 可當先手，每隻 mob只能使用五次。中毒者會隨著心跳慢慢的扣血，只能丟火玫瑰毒和深宮奇毒。
製毒 cmd make poison <原料的名字>

一刻香：天冰奇毒 (coldpoison) Lv 10，

原料：一枝花煙草

火玫瑰毒：天冰奇毒 (coldpoison) Lv 40，

原料：一朵火玫瑰花

深宮奇毒：天冰奇毒 (coldpoison) Lv 60，

原料：一顆毒龍心

五毒：天冰奇毒 (coldpoison) Lv 80，

原料：龍王心、青花蛇、千毒鳥的蛋、百年寒玉花、金線蟲各一個

１. 做出來的毒可以用來吸或是供 Per 使用。

２. 材料可以找魔園裡的毒販買，龍王心買不到，要去毒園某處山洞裡面用打的。

３. 魔教裡面很多 mob 身上都有現成的毒，用打的比較快。
