條件:

　　

1. 五種一階法術 Lv 30

  　　

解法:

【感謝 (yeah) 提供以下解謎過程】

　 

【先到 re 8e 3s e enter e n w s e out 5e se ne n 2e ne n 3e n 2e pass 2n 3e ne 3nu enter 5n，找到道尊】

　道教先天　 道尊(Master dao)

　　　 

【然後 s 2e，找到元修子】 

道臨 元修子(Taoist yuan)
　　

【ask yuan about 風水堪輿之術】

你恭敬地向元修子請教關於「風水堪輿之術」的問題。
   
元修子(taoist yuan)低頭沉思.
元修子說道：「你去拿個水晶寶鏡來給我我就教你」

　  

【然後 (waveto 2) 2n enter 2n e，找到右侍衛】

右侍衛(Right guard)

　   

【然後把右侍衛幹掉撿他屍體裡的東西】 

        　　　　手中拋出一張咒符，藉著咒符吸取天地正氣　　　　

▄▄▄▄▄▄▄▄▄▄▄▄　是故易有太極。是生兩儀。兩儀生四象　▄▄▄▄▄▄▄▄▄
█◤　▆▆▆　　　　◥　　四象生八卦。八卦定吉凶。吉凶生大業　　◤
◥　　▆  ▆　　　你左踏東方，右踩正南，真氣鼓動，信手拈來一招《離  火》
　　　▆▆▆　　　霎時火光大閃，頗有『火耀千秋怒若濤、隻手旋天即所離』之感　　◤
結果右侍衛被你以勁力一震，胸口有如受到一記重錘，連退了五六步﹗
( 右侍衛看起來可能受了點輕傷。 )
   
右侍衛死了。
   
你得到 0 點實戰經驗。
你得到 0 點潛能 1 點評價。

  

【get mirror from corpse】 

你從右侍衛的屍體身上搜出一件水晶寶鏡。
  
【然後回到元修子那，give yuan mirror】
你給元修子一件水晶寶鏡。
  
元修子拿出丹青在鏡面上畫了畫，做出一個羅盤，並教授你如何使用《風水堪輿之術》。
  
【風水堪輿之術可以設定地氣，地氣屬性可生法術屬性則法術屬性威力加強，反之則弱】
【打怪的時候風水堪輿之術就會增加熟練度囉！】
【使用方法，cmd set [gold/wood/water/fire/earth/all]】
　
           金生水剋木　　　　若是施放金屬法術則，cmd set earth
           木生火剋土　　　　　　　　木屬法術則，cmd set water
           水生木剋火　　　　　　　　水屬法術則，cmd set gold
           火生土剋金　　　　　　　　火屬法術則，cmd set wood
           土生金剋水　　　　　　　　土屬法術則，cmd set fire
　　　　　　　　　　　　　　　　　　　　　　　　　 ，cmd set all，則生金木水火土
