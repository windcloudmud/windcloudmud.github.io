金刀門公會指令介紹
名稱 指令 條件

備註
霸者之威 cmd bakee <id> Exp > 500萬

1. 用霸氣震攝敵人，增加霸氣值 ，有各種效果

2. 只能非戰鬥中使用，無法當先手

3. 霸者之威等級越高，成功機率越高

4. 霸者之威等級越高，霸氣值增加越多

5. 對方 Exp < 250萬 無法使用

6. 霸氣值上限為自身經驗值的兩倍

霸者之威功能詳細介紹
成功 / 失敗 效果
失敗 增加少量霸氣值並遭到對方主動攻擊
只增加少量霸氣值
成功 增加霸氣值並使對方忙碌
增加霸氣值並降低對方內力
增加霸氣值並使對方受到傷害
增加霸氣值並使對方丟掉身上所有物品逃跑，且得到少量經驗值
增加霸氣值並使對方直接暈倒
